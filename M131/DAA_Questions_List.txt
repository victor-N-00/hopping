DAA MASTER QUESTION LIST (Questions + Approach)

================ ASSIGNMENT 1 ================

Q1. Egg Dropping (Infinite Floors)
Problem:
Find the highest floor h such that an egg does not break when dropped.
Constraint: O(log h) drops.
Method:
Exponential search (1,2,4,8...) to find range, then binary search.

--------------------------------------------------
Q2. Minimum Length Subarray ≥ S
Problem:
Given array a[1..n], find shortest subarray with sum ≥ S.
Method:
Prefix sum + binary search OR sliding window (O(n)).

--------------------------------------------------
Q3. Median of Two Sorted Arrays
Problem:
Find median of two sorted arrays of size n.
Method:
Binary search partition (O(log n)).

--------------------------------------------------
Q4. Pair Sum = S
Problem:
Find two numbers whose sum = S.
Method:
Sort + two pointers (O(n log n)).

--------------------------------------------------
Q5. Power Form Check
Problem:
Check if a = b^k (k>1).
Method:
Try k from 2 to log(a), use binary search on b.

--------------------------------------------------
Q6. Land Redistribution
Problem:
Find max feasible f and corresponding c.
Method:
Binary search on f, adjust c using prefix sums.

--------------------------------------------------
Q7. Treasure Hunt
Problem:
Find unknown N minimizing distance.
Method:
Alternate exponential search: +1,-2,+4,-8...

--------------------------------------------------
Q8. Recurrences
Solve:
1. T(n)=T(n/2)+n^3 → O(n^3)
2. T(n)=4T(n/3)+n → O(n^log3(4))
3. T(n)=4T(n/2)+n → O(n^2)
4. T(n)=3T(n-1)+1 → O(3^n)
5. T(n)=16T(n/4)+n^2 → O(n^2 log n)
6. T(n)=4T(n/2)+n^2 log n → O(n^2 log n)

--------------------------------------------------
Q9. Max Profit
Problem:
Buy once, sell once for max profit.
Method:
Track minimum price so far.

--------------------------------------------------
Q10. Celebrity Problem
Problem:
Find person known by all but knows none.
Method:
Two-pointer elimination.

--------------------------------------------------
Q11. Polynomial Evaluation
Problem:
Evaluate P(x).
Method:
Horner’s method.

--------------------------------------------------
Q12. Function Mapping Subset
Problem:
Largest subset where mapping is bijection.
Method:
Find cycles in graph.

--------------------------------------------------
Q13. Interval Containment
Problem:
Count intervals not contained in others.
Method:
Sort by left asc, right desc.

--------------------------------------------------
Q14. k-Subset Product Sum
Problem:
Sum of all k-size subset products.
Method:
DP / combinatorics.
